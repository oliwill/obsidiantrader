# /learn - 从反馈学习

## 何时使用
积累一定分析历史后，进行自我优化

## 目标
提炼有效模式，持续改进分析质量

## 学习流程

### 1. 数据分析
```python
# 查询历史数据
- 高评分分析的共同特征
- 低评分分析的失败原因
- 用户反馈的关键词
```

### 2. 模式识别
```python
# 识别有效模式
- 哪些分析方法在特定场景下效果更好？
- 哪些股票特征预示更好的分析准确度？
- 用户的偏好是什么？
```

### 3. 模型调整
```python
# 优化策略
- 调整评分权重
- 增加/减少某些指标
- 改进风险提示逻辑
```

### 4. 知识沉淀
```python
# 更新技能文档
- 记录新发现的有效模式
- 更新常见问题排查指南
- 完善分析方法选择建议
```

## 学习输出
```
本次学习发现:
1. 有效模式: xxx (成功率 xx%)
2. 改进建议: xxx
3. 待验证假设: xxx
```
