# /check - 分析结果检查

## 何时使用
完成分析后，输出结果前

## 目标
- 验证分析逻辑的合理性
- 检查数据异常
- 确保风险提示到位

## 检查清单

### 1. 数据验证
- [ ] 价格数据是否在合理范围？
- [ ] 财务指标是否与行业平均水平偏离过大？
- [ ] 时间序列是否连续？

### 2. 逻辑检查
- [ ] 分析结论是否与数据一致？
- [ ] 多空观点是否平衡？
- [ ] 是否存在幸存者偏差？

### 3. 风险提示
- [ ] 是否说明了分析的局限性？
- [ ] 是否提及了重大不确定性？
- [ ] 是否强调了不构成投资建议？

### 4. 可解释性
- [ ] 关键结论是否有数据支撑？
- [ ] 技术术语是否解释清楚？
- [ ] 评分依据是否透明？

## 自检输出
```
✓ 数据验证通过
⚠ 需要关注: xxx
✗ 发现异常: xxx
```
