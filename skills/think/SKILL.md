# /think - 分析前思考

## 何时使用
开始任何新的股票分析前

## 目标
- 明确分析目标和边界
- 识别潜在盲点
- 选择合适的数据源和分析方法

## 检查清单

### 1. 理解需求
- [ ] 用户想要基本面、技术面还是综合分析？
- [ ] 时间范围是否明确？
- [ ] 这只股票属于什么行业/市场？

### 2. 数据准备
- [ ] 股票代码是否有效？
- [ ] 数据是否足够？（新股、停牌等情况）
- [ ] 需要补充哪些外部数据？

### 3. 分析方法选择
- [ ] 根据股票类型选择模型（成长股、价值股、周期股）
- [ ] 是否需要特殊处理？（ST股、次新股、外资股）
- [ ] 有哪些已知的风险因素？

### 4. 输出规划
- [ ] 用户需要什么样的结论？
- [ ] 需要哪些可视化图表？
- [ ] 风险提醒是否充分？

## 输出格式
```
思考摘要:
- 分析类型: xxx
- 数据状态: xxx
- 方法选择: xxx
- 潜在风险: xxx
```
